xlsuite by xlsuite

CRM, CMS, ERP | Integrated business management platform built in Ruby with a an ExtJS based administrative area.

Quickly deploy an integrated system with client and group management features, blogs, shopping cart, mass mail, access control, scripted action handler for scheduled tasks. Install a trial suite at http://XLsuite.com to see it in action

Dependencies

Many Gems

Install

XLsuite is built on-top of MySQL and cannot currently use any other database servers. 1. cp config/database.yml.sample config/database.yml 2. Edit config/database.yml to suit you (must use MySQL) 3. rake db:create db:structure:load db:bootstrap 4. script/server 5. http://localhost:3000/
